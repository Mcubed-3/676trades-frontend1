EA Setup Guide

Install the 676Trades EA in MetaTrader 5 and connect it to your dashboard.

Supported markets

The current public version supports:
  • Gold: XAUUSD
  • Major FX: EURUSD, GBPUSD, USDJPY
Crypto & indices are temporarily locked while dedicated strategies are developed.

Step 1 — Download the EA

Download the EA file from this page:
Download EA (BinaryScalper.ex5)
If the link doesn’t work yet, upload the file to your frontend hosting (or a private link) and update this URL.

Step 2 — Install MetaTrader 5

  1. Install MetaTrader 5 on a Windows PC (or Windows VPS).
  2. Login to your broker account inside MT5.

Step 3 — Install the EA into MT5

  1. Open MT5
  2. File → Open Data Folder
  3. MQL5 → Experts
  4. Copy the EA file into the Experts folder
  5. Restart MT5

Step 4 — Allow WebRequest

  1. MT5 → Tools → Options → Expert Advisors
  2. Enable “Allow WebRequest for listed URLs”
  3. Add:
https://api.676trades.org

Step 5 — Attach the EA to a chart

  1. Open a chart (example: XAUUSD on M5)
  2. Navigator → Expert Advisors → drag the EA to the chart
  3. Paste your API key into the EA input called API_KEY
  4. Turn on Algo Trading in MT5
Your API key is on Dashboard → Copy API Key. Do not share it.

Step 6 — Confirm connection

In Dashboard you should see: EA Connected ✅. If it says Invalid API key, copy again from Dashboard and paste exactly.
