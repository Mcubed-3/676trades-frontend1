EA Setup Guide

Follow these steps to install the 676Trades EA in MetaTrader 5 and connect it to your dashboard.

How 676Trades works

The EA runs inside MetaTrader 5 on your own computer or VPS. Your 676Trades dashboard sends your settings to the EA, and the EA executes trades on your broker account through MT5.

What you need

  • A Windows PC (or a Windows VPS)
  • MetaTrader 5 installed
  • A broker account that supports MT5
  • Internet connection
  • Your 676Trades API key (from Dashboard → Copy API Key)

Step 1 — Install MetaTrader 5

  1. Download MT5 from the official website.
  2. Install MT5 on your Windows PC (or VPS).
  3. Login to your broker account inside MT5.

Step 2 — Install the 676Trades EA

  1. Open MT5
  2. Click File → Open Data Folder
  3. Open MQL5 → Experts
  4. Copy the EA file into the Experts folder
  5. Restart MT5
If you do not have the EA file yet, download it from your dashboard or the link provided by 676Trades support.

Step 3 — Allow WebRequest

  1. In MT5 go to Tools → Options → Expert Advisors
  2. Enable “Allow WebRequest for listed URLs”
  3. Add this URL:
https://api.676trades.org
If you do not add this URL, the EA cannot communicate with the 676Trades backend.

Step 4 — Attach the EA to a chart

  1. Open a chart (example: XAUUSD on M5)
  2. Find the EA in Navigator → Expert Advisors
  3. Drag the EA onto the chart
  4. Paste your API key into the EA input called API_KEY
  5. Turn on Algo Trading in MT5
Your API key is in Dashboard → Copy API Key. Do not share your key.

Step 5 — Confirm connection

  • In 676Trades Dashboard you should see: EA Connected ✅
  • In MT5 check the Experts tab for messages like: Backend ok
If you see Invalid API key, copy the key again from your dashboard and paste it into the EA input exactly.

PC vs VPS (24/7 trading)

If you run MT5 on your personal computer, trading only runs while your computer is ON, MT5 is open, and the internet is connected. For uninterrupted 24/7 uptime, many users choose a VPS.
Important: 676Trades does not sell VPS services and is not affiliated with any VPS provider.

Mobile usage (phone / tablet)

The EA does not run on mobile. The EA runs inside MT5 on your PC or VPS. You can still use the MT5 mobile app to monitor trades and account activity.
