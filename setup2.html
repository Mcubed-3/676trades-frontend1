EA Setup Guide

Follow these steps to install the 676Trades EA in MetaTrader 5 and connect it to your dashboard.

How 676Trades works

The EA runs inside MetaTrader 5 on your own computer or VPS. Your 676Trades dashboard sends your settings to the EA, and the EA executes trades on your broker account through MT5.

What you need

  • A Windows PC (or a Windows VPS)
  • MetaTrader 5 installed
  • A broker account that supports MT5
  • Internet connection
  • Your 676Trades API key (Dashboard → Copy API Key)

Step 1 — Download the EA

Download the EA file and keep it somewhere easy to find (for example: Downloads folder).
Download EA (BinaryScalper.ex5)
If the download fails, it usually means the file is not hosted yet. You must upload BinaryScalper.ex5 into your frontend repo at: /downloads/BinaryScalper.ex5

Step 2 — Install MetaTrader 5

  1. Install MT5 on your Windows PC (or VPS).
  2. Login to your broker account inside MT5.

Step 3 — Install the 676Trades EA into MT5

  1. Open MT5
  2. Click File → Open Data Folder
  3. Open MQL5 → Experts
  4. Copy the downloaded BinaryScalper.ex5 into the Experts folder
  5. Restart MT5

Step 4 — Allow WebRequest

  1. MT5 → Tools → Options → Expert Advisors
  2. Enable “Allow WebRequest for listed URLs”
  3. Add this URL:
https://api.676trades.org

Step 5 — Attach the EA to a chart

  1. Open a chart (example: XAUUSD on M5)
  2. Navigator → Expert Advisors → drag BinaryScalper onto the chart
  3. Paste your API key into the EA input called API_KEY
  4. Turn on Algo Trading in MT5

Step 6 — Confirm connection

  • In Dashboard you should see: EA Connected ✅
  • In MT5 → Experts tab: you should not see Invalid API key

PC vs VPS (24/7 trading)

The EA runs only while MT5 is open and connected. For uninterrupted uptime, many users choose a VPS (optional).
676Trades does not sell VPS services and is not affiliated with any VPS provider.

Mobile usage (phone / tablet)

The EA does not run on mobile. Use the MT5 mobile app to monitor trades/account activity.
